After I have finished constructing my blog with hexo, the default theme is landscape so that I decided to change the theme to a more beautiful one. :)

I have met many problems changing the theme to diaspora so that I would like to summary it and let others have a better time changing your own theme.

Choose Theme

Choose one of the theme you like here.

Find the github address in the website. For, diaspora, it is Here @Fechin!

Follow the readme.md to do the instructions.

Instructions

  1. Clone the theme.

    git clone https://github.com/Fechin/hexo-theme-diaspora.git diaspora

tip:
Open Git Bash under hexo –> enter command “cd themes“ then clone the theme.

  1. Modify _config.yml of hexo.

    theme: diaspora

tip:
Pay attention to the blank after the colon.

  1. Change the theme

    git pull

  2. Create a new blog.

    hexo new post "yourpostname"

  3. Write a new blog.

tip:
We modify the cover of each article here and put the source file under img folder of diaspora.

Exception1 : Cannot read property ‘indexOf’ of null.
Solution1 : The link of the title cannot be null even if you don’t have it.

Exception2 : end of the stream or a document separator is expected at line x, column y.
Solution2 : Pay attention to the blank. There might be missing blanks.

  1. Config gitalk.

    Register a new application here.

Tip:
The Homepage URL is the address of your blog.

Pay attention that the Authorization callback URL should be the address of your blog.

Copy the Client ID and Client Secret and modify the _config.yml. 

clientID: 'yourclientID'
clientSecret: 'yourclientSecret'
repo: 'XinhaoZhou123.github.io'
owner: 'XinhaoZhou123'
admin: ['XinhaoZhou123']

  1. Generate and deploy your blog!

    hexo g AND hexo d
