Build Blog Using Github+Node.js+Hexo

十月 27, 2018
  • git

I have been working on building my own blogs for several days and have met several mistakes and problems. I have searched a lot for help. Unfortunately, most of the blogs partially fits my situation so that I would like to write a blog to help people who have the same situation with me to enjoy a relatively easy time with building your own blog.

Git

Download the git suitable for your own computer.

Download Git

Follow the instruction and setting the default value is OK.

Test whether you have successfully install git.

open cmd –> type command “git version”

OR

open Git Bash –> type command “git version”

Github

Sign up for github.

Sign Here.

Remember your username, password, email.

Create a new repository named yourgithubname.github.io

Copy the ssh of your repository.

Enter command ssh-keygen -t rsa -C “youremail”

Follow the instruction and we will get two more files in the folder under the user folder of C Drive.

Add your ssh key to github. Copy the info in id_rsa.pub to github.

Settings –> SSH and GPG keys –> New SSH key.

Open Git Bash –> enter command “ssh -T git@github.com“ to test your ssh key setting.

Node.js

Download node.js suitable for your computer.

Download here.

Install node.js and Setting default is OK.

Open cmd/Git Bash –> Enter command “node -v“ –>Enter command “npm -v“

Hexo

Create a folder to hold hexo.

Tip: Be carefully when you put it in C Drive because you meet several problems about permission.

open Git Bash –> enter command “npm install hexo -g“ –>enter command “npm install hexo-cli -g“ –> enter command “npm install hexo –save“ –> enter command “hexo init”

Tip: The command “git init” should be entered under the folder containing hexo.

enter command “hexo s”–> open the browser and enter “localhost:4000“ to test hexo.


(get this picture from here.)

Open Git Bash under hexo folder –> enter command ‘hexo new poster “yourblogname”‘ –>enter command “hexo g“ –> enter command “hexo d“ –> open browser enter “yourgithubname.github.io“ to browse your blog.
